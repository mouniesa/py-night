Challenge: Quint keeps his shark tracking notes in a secret log file. Read the file to find his codeword.

Task:

  1. Open and read 'captains_log.txt'.
  2. Find the codeword inside.
  3. write a script that prints the codeword exactly as written.
  4. If correct, you'll see the passphrase. Submit that passphrase.

Note: The file 'captains_log.txt' will be in the same folder as this script.
