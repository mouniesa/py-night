Amity Island — Code Lock
“We're gonna need a bigger boat.”
Challenge: The shark tracker is protected by a MasterMind-style code lock. The secret code is a sequence of 4 digits (0–9). Break it before 5000 tries… or we’re locked out.
Harbor Control • Secure Access
Rules & Task:
  • After each guess, you receive a history of (guess, (correct positions, correct digits wrong place)).
  • Example history: [([0,0,0,0], (2,0)), ([0,0,1,1], (3,0)), ([0,0,1,6], (3,0))]
  1. Write mastermind_guesser(history) that returns your next 4-digit guess as a list.
  2. Keep guesses consistent with the feedback in history.
