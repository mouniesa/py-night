Amity Island Sonar Station
“Farewell and adieu to you fair Spanish ladies…” — Quint
Challenge: Quint’s sonar picked up a reading, but it’s encoded. You need to retrieve it from the sonar system and decode it.
Amity Police • Harbor Patrol
Task:
  1. Write a script that calls sonar_reading() to get the base reading.
  2. Multiply it by 3, then add 2 to get the depth code.
  3. If the depth code is even put A in front, otherwise put B.
  4. Return that depth code.

Example: If sonar_reading() returns 12, the depth code would be A38.
